Sell eSIMs for digital delivery with WooCommerce and DataPlans.io

== Description ==

To use this plugin, you must first have a free account at [Dataplans.io](https://esims.dataplans.io/signup "Signup").  This plugin requires WooCommerce.

With this plugin you can add and resell all the Dataplans.io eSIM inventory.  Make the product, choose your markup and digitaly sell your eSIM.


Some Plugin details:

* Connect to your DataPlans account via your API key
* Start with the sandbox environment to text your payment gateways
* Switch to live environment for live purchases
* Use the Autocomplete WooCommerce Orders plugin to make instant deliveries

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'DataPlans'
3. Activate DataPlans eSIMs for WooCommerce from your Plugins page.

= From WordPress.org =

1. Download DataPlans eSIMs for WooCommerce.
2. Upload to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate DataPlans from your Plugins page.

== Frequently Asked Questions ==

= Where can I see a list of the current eSIMs I can sell? =

Our most up to date inventory will be displayed in the [online account](https://esims.dataplans.io/ "Login").

= Where can I see pricing? =

Pricing is listed in your account.  Any eSIMs sold are deducted from your account balance.
